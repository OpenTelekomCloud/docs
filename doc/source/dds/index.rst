==========================
Document Database Service
==========================


Document Database Service (DDS) is a cloud-based NoSQL database service. It features high-performance storage, high-availability architecture, and primary/standby switchover. Additionally, DDS provides online capacity expansion along with backup and restore capabilities. It has a mature performance monitoring system, a multi-level security protection mechanism, and a professional database management platform.
