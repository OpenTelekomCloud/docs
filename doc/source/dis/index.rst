=======================
Data Ingestion Service
=======================


Data Ingestion Service (DIS) serves as a management platform that captures and stores a large amount of data in real time. With its flexible data collection, efficient data transmission, and real-time data distribution features, you can easily build applications capable of processing or analyzing streaming data.
