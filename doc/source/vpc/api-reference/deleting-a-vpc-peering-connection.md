# Deleting a VPC Peering Connection<a name="vpc_peering_0007"></a>

## Function<a name="section9999113915104"></a>

This API is used to delete a VPC peering connection.

A VPC peering connection can be deleted either by the local or peer tenant.

## URI<a name="section129990396107"></a>

DELETE /v2.0/vpc/peerings/\{peering\_id\}
