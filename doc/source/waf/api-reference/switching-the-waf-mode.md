# Switching the WAF Mode<a name="EN-US_TOPIC_0193631127"></a>

## Function Description<a name="section15884400"></a>

## Examples<a name="section1641724123619"></a>

**protect\_status**  with a value of  **1**  is used as an example.

-   Request example

    ```
    {
        "protect_status": 1
    }
    ```


-   Response example

    ```
    {
        "protect_status": 1
    }
    ```
