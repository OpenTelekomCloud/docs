## Examples<a name="section140302775820"></a>

-   Request example

    ```
    {
      "path": "/login",
      "category": "params",
      "index": "name"
    }
    ```


-   Response example

    ```
    {
          "id": "e1c0e55865544d1f8c95cf71df108c6b",
          "policy_id": "yuc0e55865544d1f8c95cf71df108c6b",
          "path": "/login",
          "category":"params",
          "index": "name",
          "timestamp": 123243414132
    }
    ```
