## Examples<a name="section15246722163414"></a>

A certificate named  **cert\_a**  is used as an example.

-   Request example

    ```
    {
     "name": "cert_b"
    }
    ```


-   Response example

    ```
    {
        "id": "388a7789d55b41d1918b3088a8f1e7f3",
        "name": "cert_b",
        "expireTime": 1565467166765,
        "timestamp": 1545467166765
    }
    ```
