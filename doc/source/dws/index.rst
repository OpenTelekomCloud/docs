=======================
Data Warehouse Service
=======================


Data Warehouse Service (DWS) is a high-performance, fully-managed massively parallel processing database service that supports row store, column store, one-click deployment, and high-speed data import. With DWS, cloud users can mine, store, and analyze massive volumes of data in a secure, reliable, and quick manner.
