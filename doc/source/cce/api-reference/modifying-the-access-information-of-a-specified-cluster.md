## Function<a name="s6fbafdbe1fae41f0bcf8051f0b7e9bd7"></a>

**Example request:**

-   Binding an elastic IP address to the cluster

    ```
    {
        "spec": {
            "action": "bind",
            "spec": {
                "id": "0ef26920-3527-405d-a7b4-27106618c2d7"
            }
        }
    }
    ```

-   Unbinding an elastic IP address from the cluster

    ```
    {
        "spec": {
            "action": "unbind"
        }
    }
    ```
