# Listing APIResources of GroupVersion autoscaling/v1<a name="cce_02_0195"></a>

## Function<a name="section20847448"></a>

## URI<a name="section53409307"></a>

GET /apis/autoscaling/v1

## Request<a name="section10921718"></a>
