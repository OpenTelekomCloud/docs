# Querying Metadata of a Stack Resource<a name="EN-US_TOPIC_0084581296"></a>

## Function<a name="en-us_topic_0057973132_section48978414"></a>

## URI<a name="en-us_topic_0057973132_section38152543"></a>

## Request Example<a name="en-us_topic_0057973132_section30134562"></a>

```
GET /v1/95d02433133a4c0a87ba6967474a2ad3/stacks/HeatStack/c89c4bb3-96cb-4a55-aafa-076a7939a306/resources/my_instance/metadata
```
