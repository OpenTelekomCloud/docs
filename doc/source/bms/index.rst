==================
Bare Metal Service
==================

A Bare Metal Server (BMS) is a physical server that provides high computing performance to meet requirements of key application scenarios. BMSs can be used in conjunction with other cloud services, such as VPC. With BMS, you enjoy both the stable performance provided by server hosting and high scalability offered by cloud resources.
