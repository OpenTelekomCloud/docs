# Deleting a Data Source<a name="EN-US_TOPIC_0172486233"></a>

## Function<a name="section11410068144834"></a>

This API is used to delete a data source. This API is compatible with Sahara.

## URI<a name="section4721807314497"></a>

-   Format

    DELETE /v1.1/\{project\_id\}/data-sources/\{data\_source\_id\}
