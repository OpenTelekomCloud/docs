======================
Scalable File Service
======================


Scalable File Service (SFS) provides scalable and high-performance file storage that can be shared with multiple ECSs, featuring robust reliability and high availability. SFS uses standard file protocols and can be integrated with existing applications and tools. SFS performance improves as file system capacity increases.
