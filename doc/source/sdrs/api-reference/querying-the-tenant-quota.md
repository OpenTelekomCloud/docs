-   Example response

    ```
    {
        "quotas": {
            "resources": [
                {
                    "type": "server_groups",
                    "used": 10,
                    "quota": 50,
                    "min": 0,
                    "max": -1
                },
                {
                    "type": "replications",
                    "used": 1,
                    "quota": 100,
                    "min": 0,
                    "max": -1
                }
            ]
        }
    }
    ```

    Or

    ```
    { 
         "error": { 
             "message": "XXXX",  
             "code": "XXX" 
         } 
     }
    ```

    In this example,  **error**  represents a general error, including  **badrequest**  \(shown below\) and  **itemNotFound**.

    ```
    { 
         "badrequest": { 
             "message": "XXXX",  
             "code": "XXX" 
         } 
     }
    ```
