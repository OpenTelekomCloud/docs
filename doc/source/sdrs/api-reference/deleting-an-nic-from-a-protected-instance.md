# Deleting an NIC from a Protected Instance<a name="sdrs_05_0509"></a>

## Function<a name="en-us_topic_0079693002_section34649765"></a>

This API is used to delete an NIC from the specified protected instance.

## Constraints and Limitations<a name="section137521015392"></a>

-   **status**  of the protection group must be  **available**  or  **protected**.
-   **status**  of the protected instance must be  **available**  or  **protected**.
-   The primary NIC cannot be deleted.

## URI<a name="en-us_topic_0079693002_section39390935"></a>

-   URI format

    POST /v1/\{project\_id\}/protected-instances/\{protected\_instance\_id\}/nic/delete
