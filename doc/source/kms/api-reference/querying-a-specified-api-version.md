    None

-   Example response

    ```
    { 
       "version":
            {
                "id":"v1.0",
                "links":
                [
                    {
                        
                        "href":"https://kms.eu-de.otc.t-systems.com/v1.0/",
                        "rel":"self"
                    }
                ],
                "min_version":"",
                "status":"CURRENT",
                "version":"",
                "updated":"2018-09-05T08:18:05Z"
            }
    }
    ```

    or

