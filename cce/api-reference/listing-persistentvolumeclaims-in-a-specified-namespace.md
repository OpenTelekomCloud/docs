# Listing PersistentVolumeClaims in a Specified Namespace<a name="cce_02_0072"></a>

## Function<a name="s56cf46cd1088494ba3345c1672a9b59a"></a>

## URI<a name="s31d720a41bf84b65b5d3417e08ffcadb"></a>
